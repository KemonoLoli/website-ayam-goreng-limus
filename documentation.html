1. Ringkasan Sistem

Sistem manajemen operasional restoran berbasis web yang lengkap untuk "Warung Ayam Goreng Limus Regency" menggunakan:

  • CodeIgniter 3.1.13 - PHP MVC Framework
  • MySQL 8.0 - Database (Port 3307)
  • Bootstrap CoreUI Free - Admin Dashboard Theme

5. Panduan Instalasi

Step 1: Import Database

-- Jalankan di MySQL client (port 3307)
SOURCE /path/to/database_schema.sql;
SOURCE /path/to/database_schema_additional.sql;

-- Optional: Jika migrasi dari data lama
SOURCE /path/to/migration_script.sql;

Step 2: Start XAMPP

  • Start Apache
  • Start MySQL (port 3307)

Step 3: Akses Admin Panel

URL: http://localhost/prototype_project/admin/login

Default Login:

  • Username: master

7. Fitur Utama

Point of Sale (POS)

  • Category-filtered menu grid
  • Real-time cart management
  • Order types: Dine-in, Take Away, Delivery
  • Payment: Cash, QRIS, E-Wallet, Transfer
  • Auto-calculate change
  • Thermal receipt printing (80mm)

Inventory & Stock

  • Stock tracking with low-stock alerts
  • Movement history (in/out)
  • Quick adjustment modal
  • Auto-update on purchase receive

Payroll

  • Auto-generate based on attendance
  • Calculate allowances & deductions
  • Workflow: Draft ‚Üí Approved ‚Üí Paid
  • Printable salary slips

Reports

  • Sales: Daily/monthly, top products, charts
  • Stock: Current levels, low stock, value
  • Attendance: Monthly recap per employee
  • Payroll: Monthly summary, totals
  • CSV Export for sales data
